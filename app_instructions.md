This will be an admin dashboard.
a. the dashboard supports Arabic and English
b. light mode and darkmode
c. bloc
d. dio
e. app is both responsive and adaptive as it will work both on mobile and web.
f. Features:
    1- admin login
    2- admin settings where the admin can update his password and app settings (refer to the api)
    3- admin can CRUD users
    4- admin can CRUD transactions
    5- admin can view transactions with filters by date range or user.
    6- admin can generate pdf, excel reports based on filtered transactions.
